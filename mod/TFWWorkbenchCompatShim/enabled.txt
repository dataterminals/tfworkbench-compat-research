This file is the UE4SS "mod is enabled" marker (its presence enables the mod when
the folder is dropped into ue4ss/Mods without a mods.txt entry).

NOTE: for correct load order relative to TFWWorkbench, prefer enabling via
mods.txt instead, placing this mod ABOVE TFWWorkbench:

    TFWWorkbenchCompatShim : 1
    TFWWorkbench : 1

See ../README.md and ../../docs/05-known-good-and-workarounds.md.
